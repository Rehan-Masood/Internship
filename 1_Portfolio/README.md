# Portfolio Landing Page

This repository contains a single-page personal portfolio built in [Index.html](Index.html). It presents Muhammad Rehan Masood’s profile as a Django Developer Intern and includes a LinkedIn-style profile section, about section, education timeline, skills tabs, projects, and contact details.

## Overview

- Built with plain HTML, CSS, and JavaScript in one file.
- Uses custom typography, animated effects, and a dark futuristic visual style.
- Includes SEO metadata, Open Graph tags, Twitter card tags, and JSON-LD structured data.
- References the local profile image [LinkedIn photo.png](LinkedIn%20photo.png).

## Sections

- Hero banner with role, summary stats, and contact buttons.
- LinkedIn-inspired profile card.
- About me section with personal summary and quick facts.
- Education timeline.
- Skills area with backend, frontend, and tools tabs.
- Projects showcase.
- Contact section with links and a message form.
- Footer with social links and back-to-top navigation.

## How To Run

1. Open [Index.html](Index.html) in a web browser.
2. Or open the folder in VS Code and use Live Server if you want auto-refresh while editing.

## File Structure

- [Index.html](Index.html) - main portfolio page.
- [LinkedIn photo.png](LinkedIn%20photo.png) - profile image used on the page.

## Notes

- The page pulls Google Fonts and Three.js from external CDNs, so an internet connection is needed for the full visual experience.
- The contact form is present in the UI, but there is no backend handler in this file.
