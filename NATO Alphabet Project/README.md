# NATO Phonetic Alphabet Converter

A simple Python project that converts a word into its NATO phonetic alphabet code words.

## Features
- Reads the alphabet mapping from a CSV file
- Prompts the user to enter a word
- Converts each letter to its corresponding NATO code word
- Displays the result as a list

## Project Files
- main.py - Contains the main program logic
- nato_phonetic_alphabet.csv - Stores the letter-to-code mapping

## Requirements
- Python 3.x
- pandas

Install the dependency with:

```bash
pip install pandas
```

## How to Run
1. Open a terminal in the project folder
2. Run the script:

```bash
python main.py
```

3. Enter a word when prompted

Example:

```text
Enter a word: hello
['Hotel', 'Echo', 'Lima', 'Lima', 'Oscar']
```
