# Mile to Kilometer Converter

A simple Python desktop application that converts miles to kilometers using a graphical user interface built with Tkinter.

## Features

- Enter a value in miles
- Convert it to kilometers instantly
- Display the result in a user-friendly window

## Requirements

- Python 3.x
- Tkinter (usually included with Python installations)

## How to Run

1. Open a terminal in the project folder.
2. Run the following command:

```bash
python Mile_to_km.py
```

3. A window will appear where you can enter miles and click the Calculate button.

## Example

If you enter `10` miles, the program will display approximately `16.09` kilometers.

- `Mile_to_km.py` - Main application script
